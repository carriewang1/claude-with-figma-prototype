# figma-prototype 技能介绍

`/figma-prototype` 是一个让 Claude 直接操作 Figma 的技能。输入截图或网址，Claude 会自动在你的 Figma 画布中构建高保真设计稿，并生成完整的设计规范文档。

---

## 核心能力

**克隆网页 / 截图到 Figma**
输入任意 URL 或上传截图，Claude 自动提取色彩、布局和组件结构，在 Figma 中 1:1 还原。

**创建可点击交互原型**
识别页面间的跳转关系，自动为按钮和导航添加 Navigate、Overlay 等交互连接。

**生成 Markdown 设计文档**
构建完成后自动输出设计规范文档，包含色值、字体、间距、圆角和组件清单。

**同步更新文档**
在 Figma 中手动修改后，一句话触发同步，Claude 只更新有变化的部分，追加变更记录。

**自然语言编辑**
直接描述修改需求，如"把按钮改成圆角 12px"，Claude 展示方案后执行。

---

## 使用方法

1. 在对话框输入 `/figma-prototype`，描述你的需求
2. 将 Figma 画布链接发给 Claude（右键画布 → Copy link）
3. 提供截图或网页 URL
4. 等待构建完成，在 Figma 中查看成果并下载设计文档

---

## 使用技巧

- **先新建空白 Figma 文件**：Claude 需要一个可写入的画布链接，新建空白文件后复制浏览器地址栏 URL 即可。
- **截图比 URL 更精准**：目标页面需要登录或包含动态内容时，直接截图还原度更高。
- **改完 Figma 随时同步**：告诉 Claude "帮我同步更新设计文档"，即可获得最新版本的设计规范。
